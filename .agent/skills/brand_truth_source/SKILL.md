# Brand Truth Source

## Identity
- **Voice**: claro, directo, honesto
- **Purpose**: fijar la verdad de marca para que todo el sitio y ads sean consistentes

## Rules
- **Banned Words**:
  - premium
  - prémium
  - lujo
  - luxury
  - eco-luxury
  - desconectar
  - desconexión
  - reconectar
  - reconexión
  - cabaña
- **Banned Patterns**:
  - "no es X / más que X / mejor que X / a diferencia de..."
- **Must Do**:
  - definir TreePod por lo que ES (qué, dónde, para quién, qué incluye, qué es extra)
  - si no es verificable, no se escribe

## Brand Facts Template
- **What Is**: Domos geodésicos en Valle Las Trancas, rodeados de bosque nativo.
- **Why Choose**:
  - Privacidad en entorno natural
  - Domo geodésico (estructura distinta) + comodidad real
  - Tinaja de ciprés (extra, si aplica) + experiencia de agua caliente
- **Includes/Extras Rule**: Siempre separar: 'Incluye' vs 'Extras' (tinaja, desayuno, etc.)

## Outputs
1. **brand_facts** (lista corta)
2. **banned_words_check** (true/false + ocurrencias)
