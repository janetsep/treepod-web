# Site Coherence Auditor
Audita coherencia total: claridad, repetición, iconos, UX fricción, SEO on-page y consistencia.

## Checks

### 1. Clarity (10s Test)
- **Qué es**: Confirmar identidad (glamping domos).
- **Dónde está**: Ubicación clara.
- **Qué incluye**: Diferenciar de extras.
- **Desde cuánto**: Precio o cálculo claro.
- **Cómo reservar**: CTA visibles.

### 2. No Repetition
- **Rule**: Una idea = un hogar. Si se repite, fusionar y recortar.

### 3. Icon Coherence
- **Banned Icons**: casa, edificio, ciudad, techo a dos aguas
- **Allowed Icons**: círculo/domo, montaña, hoja, fuego/calor, agua/tinaja, wifi
- **Rule**: Icono crítico siempre con etiqueta de texto.

### 4. UX Anti-Friction
- Evitar piso falso.
- CTA visible sin scroll excesivo (mobile).
- CTA secundario visible.

## Outputs
- **top_issues** (10, severidad ALTA/MEDIA/BAJA)
- **before_after_fixes** (texto exacto)
- **repetition_map** (qué borrar)
- **icon_table** (icono | dónde | problema | recomendación)
