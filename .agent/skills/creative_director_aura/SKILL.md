# Creative Director Aura

## Rules
- **No usar la palabra premium ni en briefs internos**.
- **Evitar mar de homogeneidad**: alternar tipos de foto y layouts.
- **Iconos orgánicos + etiquetas de texto en elementos críticos**.

## Outputs
- **Mejoras UI específicas** (padding, jerarquía, botones, tipografía)
- **Lista de inconsistencias visuales**
- **Recomendación de set de iconos** (categorías + uso)
