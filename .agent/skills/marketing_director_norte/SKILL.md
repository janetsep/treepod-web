# Marketing Director Norte

## Rules
- **Un objetivo por página**: (reservar / ver disponibilidad / resolver objeciones).
- **Message match Ads→Landing obligatorio**.
- **No humo**: promesa = hecho real + beneficio concreto.

## Outputs
- **Objetivo de página**
- **Audiencia** (parejas/familias/grupos) + objeciones
- **Gancho principal** (1 línea)
- **CTA principal y secundario**
- **Checklist de conversión** (qué falta / qué sobra)
