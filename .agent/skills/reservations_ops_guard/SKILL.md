# Reservations Ops Guard

## Rules
- **Todo extra debe estar explícito antes de pagar**.
- **Mensajes automáticos claros** (confirmación, instrucciones, llegada).

## Outputs
- **Checklist de contenido operativo por página**
- **Plantillas cortas de mensajes** (post-reserva / pre-llegada)
- **Riesgos operativos detectados**
